Units aims to improve your browsing experience by helping you access the most important parts of a website quickly and easily. Here are a few important concepts to help you get started..

Content Units (CUs)

Search results pages, social network feeds, Q&A sites, news sites and blogs: all commonly present the primary content on the page as distinct 'units' or blocks. We call these blocks "Content Units".

  • To see an example, visit one of: Google Search, Facebook, Twitter or Stack Overflow.
  • Notice that the first CU on the page is already selected. Press j to select the next CU.
  • Similarly, you can press k to select the previous CU.
  • When you select a CU, the "main element" in it is focused. You can open this element in a new tab using o or in the same tab using enter. Pressing the tab key moves the focus forward to the next element.

Now that we have found these CUs on our favorite sites, there is a lot of fun features we can enable using them. Interested? Explore more "CU based" shortcuts!

General Utilities

To enable a seamless unit-y experience across all websites, this extensions gives you a few awesome general utilities.
  • On pages that don't have any CUs, j and k scroll the page down and up. Check out how smooth it is :)
  • Quickly access any link on the page. Press space and letter to reach a link starting with that letter.
  • Other shortcuts included for navigating quickly to different parts and elements of a page, between tabs, closing tab, and more.
