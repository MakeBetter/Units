UnitsBeta

Browse the web with greater ease and focus.

A Chrome extension for an enhanced browsing experience, based on effortless navigation between the main Units of a page. Includes wins for productivity, focus, and accessibility along the way.

Demo

Key Features

The core idea here is that we identify the main 'units' of content on a website.

What are these units? For a Google search page, these would be the search results. For Facebook, the shared posts. For a blog, these are the individual posts. For a Q&A or a forum, the units would be yes, the question and its related answers.

Once we find these units on a page, we can use them to do many very cool things that help focused browsing, productivity, usability and accessibility (and more!). We'd like to show you some of the features that we've been working on:

Scrolling done right

Scrolling is the most common action performed on the web! There is a certain beauty to scrolling by units -- it combines the strengths of a slide show (which allows viewing content in discrete logical chunks) with those of a continuous scrollable page.

Through feedback and iterations, we've tried to perfect the scrolling experience across different websites and different types of units. Recommend that you try the extension and check it out.

Try it out:

Press 'j' and 'k' to scroll this page in Units mode.

Focused reading

Spotlight Effect

Dim out the rest of the page, and read one post at a time with focus. Try it!

Spotlight effect on the selected unit on Quora. Press "+" to increase the dimming effect and "-" to decrease it.

Filter Units

The smart "find" feature you did not realize you needed. It finds units by your search term and hides all the other units on the page.

For pages that have infinite scroll, it's even more useful.

Consistent shortcuts across sites

Units enables consistent shortcuts that work in the context of the selected unit. For example, I can press "u" to upvote a post, "s" to share a post, "m" to see more, etc.

The cool thing about this is that the same shortcuts work consistently across the supported websites. It helps makes these shortcuts so much easier to use and remember.

Supported Sites

Units is supported on a fairly large percentage of the most popular websites.

  • Social networking sites
    : Facebook, Twitter, LinkedIn
  • Search Engines
    : Google, HN Search
  • Q&A sites
    : Quora, Stack Overflow, All StackExchange sites
  • Blogs and Microblogs
    : Many Tumblr and Blogspot blogs (we working on supporting more), GigaOm
  • News
    : Hacker News, NY Times
  • E-commerce, Classifieds
    : Amazon, Craigslist
  • Others
    : YouTube, Github

Adding support for a website is very easy and quick. You would just need add the data for your site URL. We'd love your help in adding more sites to the repository!

Minimal Interference

To provide an integrated browsing experience across websites, Units needed to work on all websites. But at the same time, we really try to be minimally invasive.

  • 1) Quickly disable extension temporarily or permanently on a webpage.
  • 2) All general shortcuts can be easily changed from the "Options" page.
  • 3) Protected shortcuts
