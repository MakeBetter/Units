UnitsBeta

Browse the web with greater ease and focus.

We identify the main units of content on a web page.

The extension provides an enhanced browsing experience based on effortless navigation between these units of a page. It packs several features for productivity, focus, and accessibility along the way.

Key Features

Scrolling done right

Scrolling is the most common action performed on the web! The is a certain quality to scrolling a web page by chunks that correspond to the logical units of content on the page. Such a way of viewing content combines the strengths of a slide show (which allows viewing content in discrete logical chunks) with the niceness of a single continuous scrollable page.

We've come up with a novel way of scrolling web pages that we think works quite well. When you press the scroll key, the page is scrolled by a unit or by a small increment, as most appropriate.

Try it out: Use 'j' to scroll this page down and 'k' to scroll up. (Works best on Chrome at the moment)

Focused reading

Dim out the rest of the page, and read one post at a time with focus.

Try it out: Press the '+' key to dim everything on the page but the selected unit.

Filter Units

The smart "find" feature you did not realize you needed. It finds units by your search term and hides all the other units on the page.

For sites such as Facebook that implement infinite scroll, this feature is even more useful.

Consistent shortcuts across sites

Units enables shortcuts that work in the context of the selected unit. For example, you can press "u" to upvote a post, "s" to share a post, "m" to see more, etc.

The cool thing about this is that the same shortcuts work consistently across all the supported websites. So, pressing 'u' will upvote the selected post on Facebook, Quora, Hacker News, Stack Overflow -- you get the idea.

Productivity goodies

In the spirit of more efficient and productive browsing, we have included a few general utilities for keyboard navigation. These work on any web page (not just those with units). These include:

  • 1) Utility for selecting any link on page with keyboard
  • 2) Shortcuts for navigating to different elements on a page
  • 3) Shortcuts for common tab operations

Supported Sites

Units is supported on a very large percentage of the most popular websites.

  • Social networking sites
    : Facebook, Twitter, LinkedIn
  • Search Engines
    : Google, HN Search
  • Q&A sites
    : Quora, Stack Overflow, All StackExchange sites
  • Blogs and Microblogs
    : Many Tumblr and Blogspot blogs (we are working on supporting more), Gigagom
  • News sites
    : Hacker News, NY Times, NPR
  • E-commerce, Classified
    : Amazon, Craigslist
  • Others
    : YouTube, Github, BackboneJS, and more!

Adding support for a website is very easy and quick. You would just need add to data for the site URL. We need your help in adding many more sites to the repository!

Minimal Interference

  • Disable Extension
    You can quickly disable/enable Units on a page.
  • Configuring Options
    Most shortcuts can be easily changed from the "Options" page.
  • Choice of shortcuts
    We don't use shortcuts that common websites provide in-built. For example, we do not use '?', '/' and 'g' based shortcuts because several websites already use these, and we wouldn't to break them.

Thanks for trying out Units! Just so you know, we do not view, transmit or store your browsing history in any way.

For more details on this, go to units.io/privacy
