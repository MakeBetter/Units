UnitsBeta

Chrome extension for focused browsing. It's good, really.

This extension is a collection of features that helps in good browsing. These features are very integrated and work well with each other.

Demo

Key Features

Content Units

Identify the most important 'units' of a webpage. Once you have these units, you can do a lot to customize your experience of web browsing.

Popular websites like social networking sites, Q&A sites, forums, news sites, all have

Scrolling done right

This might be a new paradigm for scrolling web pages.

Scrolling is the most common action performed on the web! If a page has discrete content units, there is a certain beauty to scrolling by CUs.

Try it out:

Press 'j' and 'k' to scroll this page in Units mode.
